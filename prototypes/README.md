# Prototypes

Standalone, single-file HTML explorations kept for reference. None of these are wired into the build — they're throwaway visual tests.

- `bubble-test.html` — transparent "bubble" avatar style test.
- `composite-test-v2.html` — SVG color-matching test against a reference arena image.
- `orb-cup-prototype.html` — original single-file version of the game. The working version now lives in [`src/`](../src) as a proper Vite project; this file is kept for diffing/history only.

Open any of them directly in a browser (`open prototypes/bubble-test.html`) — no build step needed.
